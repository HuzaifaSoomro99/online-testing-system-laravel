# Online Testing System (Laravel)

## Description
A web-based MCQ testing platform developed using Laravel. The system allows students to attempt timed quizzes and receive instant results, while admins can manage questions, exams, and results.

## Features
- MCQ-based tests
- Timer-based exams
- Instant results
- Admin panel (questions, exams, results)

## Tech Stack
- Laravel
- MySQL
- Blade

## How to Run
1. Clone repo
2. composer install
3. setup .env
4. php artisan migrate
5. php artisan serve
